# 中文填写版 Prompt 模板

这份文档用于手工补词和快速迭代。

使用方法：
- 把每个模板里的方括号内容替换成你的具体描述。
- 如果某一项不需要，就删掉整行，不要留空。
- 先写清楚光线和产品位置，再补人物、环境、细节。
- 每次只改一个大变量，方便比较出图差异。

推荐填写顺序：
1. 场景和时间
2. 产品位置和露出比例
3. 人物动作
4. 灯光方向
5. 环境道具
6. 负面词
