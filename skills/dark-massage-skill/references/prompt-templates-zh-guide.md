# 中文模板使用说明

推荐工作流：

1. 先选模板
- 主视觉和大场景用 `模板1` 或 `模板5`
- 办公场景用 `模板2`
- 亲子情绪图用 `模板3`
- 颈部热敷功能图用 `模板4`
- 俯视使用图用 `模板6`
- 材质卖点图用 `模板7`
- 对比说明图用 `模板8`
- 快速试风格用 `模板9`

2. 先锁三件事
- 光线：必须先写清是日落、傍晚、暖夜、落地灯，避免模型默认白天
- 产品位置：必须写清放在哪里、贴不贴身、露出多少
- 人物动作：必须写成具体姿势，不要只写“坐着”或“放松”

3. 一次只改一个大变量
- 先不要同时改场景、发型、动作、产品位置
- 推荐只改其中一个，方便找出有效方向

4. 电商图优先级
- 先保产品可读
- 再保场景氛围
- 再保人物状态
- 最后才加道具和故事感

5. 常见失败时怎么补
- 太亮：补“低照度、背景更暗、深暖阴影、不是白天”
- 太冷：补“暖琥珀、暖橙、蜂蜜色光线、禁止冷蓝调”
- 产品不明显：补“露出比例、按钮区可见、产品是画面主角之一”
- 人太僵：补“慵懒、自然靠坐、肩膀放松、手指自然弯曲”
- 太像AI：补“8K RAW、真实材质、无塑料感、无CG感、无AI smoothing”

建议你实际使用时，把中文模板填完后，直接复制成一版“完整成品 prompt”，不要把方括号留在最终输入里。
