# Call Mode — 调用模式

当用户无参考图时，使用已建立的品类视觉框架直接生成标准化提示词。

## 目的

调用模式不分析新图片。它复用已建立的品类视觉规则体系，产品外观描述由用户提供或标记为待填入。

## 最小输入

- product name（产品名）
- shot type（画面类型）
- usage scenario（使用场景）
- image ratio（画面比例）
- 任何必需的产品外观事实（颜色/材质/结构 —— 由用户参考图提供）

## 默认行为

如果用户省略某个变量，使用品类视觉框架中最保守的匹配选项。产品外观绝不编造。

## 调用模式工作流

1. 确认产品名和画面类型
2. 从预设族选择最匹配的模板
3. 锁定品牌视觉规则（人物/服装/灯光/构图/质量）
4. 产品描述填入变量占位符或用户提供的产品事实
5. 输出标准化 6 段提示词包

推荐阅读顺序：

1. `调用模式预设`
2. `模板查找目录`
3. `统一调用基线`
4. 对应品类规则（如有）
5. 选定的完整模板

## 调用模式预设

- `白底极简英雄图`: 单张产品英雄图，白底，构图极简。
- `黑蓝科技英雄图`: 深色渐变英雄图，电影感光线，科技感更强。
- `红黑性能英雄图`: 黑红性能英雄图，对比更强，销售压迫感更强。
- `白底顺滑湿剃图`: 白底或浅灰底的湿剃使用图，重点表现泡沫、贴肤、顺滑。
- `黑底高效剃须图`: 黑底高对比使用图，重点表现高效、利落、剃净。
- `科技贴合剃须图`: 深色科技使用图，重点表现贴合面部曲线和刀头接触精度。
- `内部结构爆炸图`: 剃须刀内部结构拆解图，重点表现马达、传动、内部动力逻辑。
- `刀头结构说明图`: 刀头局部结构放大图，重点表现长短刀组合与剃须逻辑。
- `刀头贴合动效图`: 刀头局部动效图，重点表现浮动、贴面、轮廓跟随。
- `刀网微距纹理图`: 刀网与中刀超微距图，重点表现精密纹理、锋利与加工精度。
- `材质触感细节图`: 材质与触感细节图，重点表现舒适圈、纹理、顺滑与亲肤。
- `机身工艺细节图`: 机身局部工艺图，重点表现全金属、边缘、logo、接口和表面处理。
- `旅行收纳图`: 便携场景图，重点表现收纳、旅行和体积小巧，不归入细节图。
- `车载出行图`: 出行场景图，重点表现车内携带与商务通勤氛围，不归入细节图。
- `防水冲洗说明图`: 功能说明图，重点表现IPX7、防水、可直接冲洗与清洁便利。
- `沉浸防水说明图`: 功能说明图，重点表现全身防水、可水洗、干湿双剃。
- `充电续航说明图`: 功能说明图，重点表现快充、长续航、充电底座或线充方式。
- `配件组合展示图`: 配件展示图，重点表现包装、配件齐全度、礼盒感和整体完整度。
- `配件清单说明图`: 配件展示图，重点表现随机配件清单、编号说明和盒内包含内容。

## 模板查找目录

先按画面任务选择大类，再进入对应完整模板：

- `KV英雄图`: 白底极简英雄图, 黑蓝科技英雄图, 红黑性能英雄图
- `使用图`: 白底顺滑湿剃图, 黑底高效剃须图, 科技贴合剃须图
- `细节图`: 先读细节图总规则，再选内部结构爆炸图、刀头结构说明图、刀头贴合动效图、刀网微距纹理图、材质触感细节图、机身工艺细节图
- `便携/出行场景图`: 先读便携/出行场景图总规则，再选旅行收纳图或车载出行图
- `功能说明图`: 先读功能说明图总规则，再选防水冲洗说明图、沉浸防水说明图、充电续航说明图
- `配件展示图`: 先读配件展示图总规则，再选配件组合展示图或配件清单说明图

选择原则：先判断这张图要完成的商业任务，再判断背景、人物、光线和道具。不要只因为画面是黑底就自动选择科技图，也不要只因为出现配件就自动选择配件展示图。

## 统一调用基线

在没有额外指定时，所有含人物模板默认继承以下人物设定：

```text
欧美成年男性，年龄约25-35岁，身形修长匀称，面部轮廓清晰，下颌线明显，五官立体但不过度夸张。发型为整洁的中短发，偏分或向后梳理，发丝自然有轻微光泽，不凌乱。整体气质冷静、克制、干净、精致，偏都市轻商务审美，不要夸张表情，不要强烈运动感，不要街头潮流感，不要粗犷硬汉风。
```

在没有额外指定时，所有含人物模板默认继承以下服装设定：

```text
以中性色高级基础款为主，如黑色高级衬衫、深灰上装、黑色高领、白衬衫、浅米色西装或其它品牌允许的低饱和简洁轻商务上装。服装应干净平整、无复杂图案、无夸张装饰、无明显潮牌街头元素。
```

在没有额外指定时，所有模板默认继承以下灯光设定：

```text
光线必须首先服务产品可读性、材质真实性和商业干净感。金属边缘与高光清楚但不过曝，暗部保留层次但不死黑，皮肤不能脏灰或塑料感。白底体系优先高键柔光或定向棚拍光，深色体系优先侧前方主光、轮廓光或受控氛围光，功能光效只能辅助说明，不能压过主体。
```

在没有额外指定时，所有模板默认继承以下负面词基线：

```text
hallucinated logo, fake text, invented structure, extra buttons, extra parts, incorrect material, incorrect color, broken proportions, warped edges, floating product, duplicated product, cluttered background, unrelated props, off-brand lighting, mixed visual language, cartoon, CGI, plastic look, oversaturated colors, unreadable product, hidden product, face dominating the frame, text-heavy collage, over-retouched skin, exaggerated cyber neon
```

配件展示图不是功能说明图，也不是细节图，而是围绕"盒内包含什么、包装体系是否完整、配件是否齐全、礼盒感是否成立"展开的单张展示型详情图。画面可以出现包装盒、手提袋、底座、线材、清洁刷、说明书、收纳盒等配件，但必须保持主次清楚，让用户快速理解"随盒都有什么"。

配件展示图视觉规则：
- 产品本体仍然是视觉锚点
- 配件围绕产品有秩序展开
- 可以更强调完整度、礼盒感、品牌包装感
- 配件组合展示图需要明确的局部定向棚拍光，产品脚下和核心配件周围必须有可辨认的椭圆形或圆形桌面受光灯斑，产品站在灯斑中心，中心产品区和近处绒布明显更亮，灯斑外快速衰减到暗部，但不要可见光柱或体积光
- 如果要逐项说明，可使用编号、标签或分区引导
- 允许更静物化，但不能变成普通杂物摆拍
- 重点是"完整展示"或"清单说明"，不是功能演示

配件展示图负面约束：
不要发明不存在的配件，不要漏掉关键配件，不要把包装体系拍得廉价杂乱，不要让配件压过产品主体，不要堆过多无关礼品，不要做成泛家居静物图，不要让编号和说明混乱难读。
